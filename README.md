# Skies — Weather Dashboard
A Flask web app that displays current weather, a 5-day forecast, and severe weather alerts for any city in the world.

## Setup
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Add your OpenWeatherMap API key to app.py
4. Run the app:
   python app.py
5. Open http://127.0.0.1:5000 in your browser

## Features
- Current weather conditions
- 5-day forecast
- Severe weather alerts
- Save frequently searched cities
- Toggle between °F and °C

## API
- OpenWeatherMap (https://openweathermap.org/api)

## Tech Stack
- Python / Flask
- HTML, CSS, JavaScript
- OpenWeatherMap API
